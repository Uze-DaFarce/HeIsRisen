# Missing Symbols and Creation Guide

The game requires exactly 60 symbols (30 Pagan, 30 Christian) to function correctly. Currently, there are 55 valid symbols (29 Pagan, 26 Christian).

## Instructions

1.  **Generate the Images:** Use the prompt below to generate the 5 missing symbol images using [Grok Imagine](https://grok.com/imagine).
2.  **Save the Images:** Save each image with the specific filename listed below in the `assets/symbols/` directory.
3.  **Verify:** Restart the game. The new symbols should appear on the eggs.

## Image Style Prompt

The game uses a specific style for symbols: a circular token with a golden Greek key pattern rim on a white background.

**Prompt Template:**

> Create a digital illustration of a [SYMBOL NAME] inside a circular golden frame. The frame should have an intricate Greek key pattern. The background inside the circle should be white. The central symbol should be [DESCRIPTION], with a clean, slightly 3D illustrative style, similar to mobile game assets. Ensure the golden rim is prominent.
